Paws City

Github Link

With life in “paws” during the pandemic, aspiring dog owners need a one tap app to find the right pet-related resources. This is an app to connect the petless with the petful. Using Petfinder and Trueway-places API’s, we provide location-based results to find your new companion, in addition to information about community parks, pet stores, and dog-friendly places to visit in your city.

Weather App

Github Link

Simple weather application, where user inputs a city, and receives current weather data with the temperature, humidity, wind speed, UV index, and an icon to correspond with the weather. The UV index number is highlighted with colors to represent the level. There is also a five-day forecast, displaying the day, temperature, humidity, and corresponding icon for each day.

Hamilton Quiz

Github Link

Test your knowledge of the hit Broadway show, "Hamilton" with this quiz. This features four questions, with a 30 second countdown timer. After a user selects an answer, a display will state whether it is correct. A final score will appear, which will display the "seconds left". The user will input their initials, and "Add to Top Scores", will record the user's intials and score into Top Scores.
